# Financial-Analysis
This report analyzes Tata Motors (automobile) and ICICI Prudential (insurance) through weekly stock performance (April 14–20, 2026) and three-year financials (FY 2021-24). It covers price movement, financials, ratios, SWOT, and outlook to assess financial health, growth, risk, and investment potential. Tata Motors is a high-growth, high-risk EV play; ICICI Prudential is a stable compounder. Data from NSE, BSE, SEBI, TickerTape, Screener.in, and company presentations.
